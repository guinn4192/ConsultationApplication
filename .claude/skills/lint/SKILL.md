CAWikiの健康診断（lint pass）を実行する。

## 手順

可能なら **`wiki-linter` サブエージェントに委譲する**（メインの会話コンテキストを汚さないため）。`Agent` ツールで `subagent_type: wiki-linter` を呼び出し、Wikiの絶対パス `C:\ConsultationApplication\wiki` を伝える。

サブエージェントが返すレポートを基に、ユーザーへ要約を提示する。

## サブエージェントが検出するもの

- **矛盾**: ページAとページBで同じ事象について異なる記述がある
- **古い記述**: より新しいソース（`updated` が新しい関連ページ）で上書きされている可能性
- **orphan ページ**: どこからもリンクされていないページ
- **概念ギャップ**: 複数ページで言及されているが独立ページが無い概念
- **横断参照不足**: 本来リンクすべき関連ページにリンクされていない箇所
- **データギャップ**: Web検索や追加ソース取り込みで埋められそうな空白

## 重要

- レポートのみ返す。**修正はユーザーの指示を受けてから**行う。
- 完了後 `wiki/log.md` に以下を追記:
  ```
  ## [<今日の日付>] lint | full pass
  - 検出: orphan N件、矛盾 N件、概念ギャップ N件、その他 N件
  - 詳細: <要約>
  ```
